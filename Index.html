Images Example:

Images are declared using the <img> tag, and specifying the source path. If the image is saved in the same folder as the HTML file, "ImageFileName.Extension" will suffice as the source. If located elsewhere, the full file path must be included.

Images can also be used directly from the internet, by right-clicking and selecting "Copy img URL", and using that as the source.

Images can be re-sized by adding the height="" and width="" declarations after the source path.

Tables Examples

<table> tags declare the scope of the table. Everything needs to be inside this to appear in the table.

<tr> tags represent a table row. <th> represents table headers and <td> represents the data for the table.

Images can also be used in a table. Same goes for text formatting and alignment.




HTML Entities:

HTML entities are just a way of writing reserved characters such as < and >. They always begin with "&" and end with ";". Click here for a quick reference of entities.
